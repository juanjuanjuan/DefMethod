Def Method Code Test

The Solution contains 2 projects. One for the actual command app, another one for the Test Units.

Running App: Open SLN location, execute: dotnet run --project .\CommandApp\CommandApp.csproj - Or just run from Command Line the following EXE .\DefMethod\CommandAppSln\CommandApp\bin\Release\net5.0\CommandApp.exe 
Running Tests: Open SLN location, execute: dotnet test
Requirements: .NET 5.0 for the Testing Cases and dotnet execution. The EXE file don't need anything.

GitHub: https://github.com/juanjuanjuan/DefMethod
